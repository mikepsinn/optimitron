# Spending KPI Levels

- Objective: estimate useful spending levels for each discretionary spending predictor.
- `Suggested Level` uses support-constrained targets, so it stays in data-backed ranges.
- `MED` is the minimum useful level where gains first become consistent.
- `Slowdown Knee` is where extra spending starts showing weaker gains.
