# Universal Pre-K (Ages 3-4)

> Federally funded universal pre-K based on Perry Preschool and European models.

| Metric | Value |
|--------|-------|
| Income impact | **+$8,931/yr** (15% of median) |
| Health impact | **+12 months** HALE |
| Evidence grade | **A** |
| Causal confidence | 88% |
| Action | reallocate |
| Category | education |

**Current status:** Only 34% of US 3-year-olds enrolled; varies wildly by state

**Target:** Federal funding for universal enrollment by age 3

## Evidence

Perry Preschool RCT: 40-year follow-up shows $7-12 ROI per dollar. France, Denmark, Finland all have universal pre-K with better educational outcomes.
