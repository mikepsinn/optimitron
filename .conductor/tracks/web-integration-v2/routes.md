# Explorer Route Contracts (v2)

## Route Map

- `/outcomes`
  - Outcome hub index.
  - Lists available outcomes and top predictor per outcome.

- `/outcomes/:outcomeId`
  - Outcome-level mega study ranking.
  - Sorted predictors with score/confidence/adjusted p-value.

- `/studies/:outcomeId/:predictorId`
  - Pair study detail page.
  - Includes summary metrics, optimal value block, adaptive bin tables, evidence/quality diagnostics, and data-flow steps.

- `/studies/:outcomeId/:predictorId/jurisdictions`
  - Subject/jurisdiction table for the selected pair.

- `/studies/:outcomeId/:predictorId/jurisdictions/:jurisdictionId`
  - Subject/jurisdiction n-of-1 detail page for the selected pair.

## Param Contracts

- `outcomeId`
  - Canonical lowercase ID generated by `toId(...)` in `packages/web/src/lib/analysis-explorer-data.ts`.
  - Must correspond to an existing `OutcomeMegaStudyRanking.outcomeId`.

- `predictorId`
  - Canonical lowercase ID generated by `toId(...)` or category-specific prefix (`budget_*`).
  - Must correspond to a pair key in `getPairStudy(outcomeId, predictorId)`.

- `jurisdictionId`
  - Canonical lowercase subject ID from `ExplorerSubjectSummary.subjectId`.
  - Must resolve with `getPairSubject(outcomeId, predictorId, jurisdictionId)`.

## Source of Truth

- Route helper module:
  - `packages/web/src/lib/analysis-explorer-routes.ts`
- Payload adapter and cache:
  - `packages/web/src/lib/analysis-explorer-data.ts`
- Shared pair-study schema validation:
